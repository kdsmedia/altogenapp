# ALTOGEN APP

Aplikasi inovasi digital berbasis AI dengan fitur manajemen finansial, sistem bonus harian, dan generator konten kreatif.

## Fitur Utama
- **AI Prompt Generator**: Membuat deskripsi adegan sinematik menggunakan OpenAI GPT-4o.
- **AI Video Generator**: Menganimasi gambar menjadi video menggunakan Replicate (Minimax).
- **Sistem Finansial**: Top Up via QRIS dinamis dan Penarikan saldo (Withdraw) ke E-Wallet.
- **Pusat Bonus**: Paket langganan premium, Check-in harian, Video rewards, dan sistem Referral.
- **Admin Panel**: Akses eksklusif untuk admin `appsidhanie@gmail.com` guna mengelola pengguna dan transaksi.

## Teknologi
- **Frontend**: Next.js 15, React, Tailwind CSS, ShadCN UI.
- **Backend**: Firebase Firestore & Authentication.
- **AI Engine**: OpenAI API, Replicate API.

## Cara Instalasi & Pengaturan Keamanan
1. Clone repositori ini.
2. Jalankan `npm install`.
3. Buat file `.env` di root folder dan masukkan API Key Anda:
   ```env
   OPENAI_API_KEY=kunci_anda
   REPLICATE_API_TOKEN=token_anda
   ```
4. Jalankan `npm run dev`.
